DEXT5 Editor :: Config :: AccessibilityValidationItems

config.AccessibilityValidationItems

웹 접근성 검증 항목 중 검증 하고자 하는 항목들을 선택하여 설정합니다.

remarks

웹 접근성 검증 항목은
"table_caption, table_summary, iframe_title, frame_title, img_alt, a_title, duplicate_id" 입니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 에디터에 작성된 테이블을 웹 접근성 검증 합니다.
        DEXT5.config.AccessibilityValidationItems = "table_caption, table_summary ";

        new Dext5editor("editor1");

    </script>       
</div>
