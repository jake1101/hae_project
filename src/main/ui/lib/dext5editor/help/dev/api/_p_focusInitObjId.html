DEXT5 Editor :: Config :: focusInitObjId

config.focusInitObjId

에디터 로드 후 설정한 아이디를 가진 객체에 포커스를 설정합니다.

remarks

필요시 DEXT5.config.focusInitObjId = "아이디값" 으로 설정합니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div border="0" style="width:900px;height:550px">    
    <script type="text/javascript">

        // 에디터 포커스를 title 아이디로 설정합니다.
        DEXT5.config.focusInitObjId = "title";

        new Dext5editor("editor1");

    </script>       
</div>
