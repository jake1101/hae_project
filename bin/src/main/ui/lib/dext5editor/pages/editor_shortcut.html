DEXT5 단축키 도움말
일반 단축키

Shift + Tab : 이전 컨트롤로 포커스 이동

ESC : 팝업 창, 글꼴 적용 리스트 창 등에서 입력 시 열려있는 창이 닫힘

Ctrl + F12 : 에디터의 단축키 도움말 팝업창 열림

이동 단축키

Ctrl + 1 : 메뉴바로 포커스 이동

Ctrl + 2 : 첫번째 툴바로 포커스 이동

Ctrl + 3 : 두번째 툴바로 포커스 이동

Ctrl + 4 : 편집 영역으로 커서를 이동

Ctrl + 5 : 하단 상태바로 포커스 이동

Ctrl + 6 : 상단 상태바로 포커스 이동

명령 단축키

Ctrl + Z : 실행 취소
Ctrl + Y : 실행 복구
Ctrl + C : 복사하기
Ctrl + V : 붙여넣기
Ctrl + X : 잘라내기
Ctrl + B : 굵게
Ctrl + I : 기울이기
Ctrl + U : 밑줄
Ctrl + D : 취소선
Ctrl + L : 왼쪽정렬
Ctrl + E : 가운데정렬
Ctrl + H : 오른쪽정렬
Ctrl + '(single quote) : 아래첨자
Ctrl + Shift + "(double quotes) : 위첨자
테이블 단축키

Ctrl + F5 : 현재 커서위치의 셀 선택
Ctrl + F6 : 현재 커서위치의 가로줄 선택
Ctrl + F7 : 현재 커서위치의 세로줄 선택
Ctrl + F8 : 테이블 전체 선택
Ctrl + F9 : 선택된 셀 병합
Ctrl + F10 : 선택된 셀의 너비를 같게
Ctrl + F11 : 선택된 셀의 높이를 같게
Ctrl + Shift + ] : 테이블 들여쓰기
Ctrl + Shift + [ : 테이블 내어쓰기
Ctrl + Enter : 1. 테이블 첫번째 셀에서 테이블 위쪽으로 한 줄 삽입
2. 테이블 마지막 셀에서 테이블 아래로 한 줄 삽입
Ctrl + Shift + C : 테이블 복사하기
Ctrl + Shift + V : 테이블 붙여넣기
Ctrl + Shift + T : 테이블 내용 붙여넣기
