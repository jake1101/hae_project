DEXT5 Editor :: Config :: mimeFileTypeFilter

config.mimeFileTypeFilter

MIME 인코딩 대상 파일을 설정합니다.

remarks

기본값은 "1"이고 "Image" 입니다.
"2"로 설정시 "css", "4"로 설정시 "js", "8"로 설정시 "동영상, swf 등 Object 파일" 입니다.

예) 이미지와 js 파일 종류 모두를 인코딩 대상으로 지정하기 위해서는
      "1" + "4" = 5 설정합니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 이미지와 js 파일 종류 모두를 인코딩 대상으로 설정합니다.
        DEXT5.config.mimeFileTypeFilter = "5";

        new Dext5editor("editor1");

    </script>       
</div>
