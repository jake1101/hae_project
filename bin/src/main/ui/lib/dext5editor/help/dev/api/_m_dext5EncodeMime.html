DEXT5 Editor :: dext5EncodeMime

string dext5EncodeMime(html, nFileType)

에디터에서 작성된 html을 MIME 형태로 인코딩 하는 기능입니다.
(단, mimeUse 설정값이"1" 이어야 합니다.)

return value

정상완료시 인코딩 소스, 오류시 0 리턴합니다.

parameters

html  인코딩할 html소스 입니다.
nFileType  파일명을 처리하는 방법을 지정합니다.
              0(false) : 파일명을 unique 한 Guid 형태로 변환합니다.
              1(true) : 원 파일명을 그대로 사용합니다.

remarks

인코딩된 데이타는 일반 폼전송방식으로 서버로 전송하면 됩니다.
인코딩된 데이타는 서버에서 받아 디코딩을 한 후 DB에 저장하여야 합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
<script type="text/javascript">

    function encodeMime() {
        // id가 editor1인 에디터 디자인 영역의 html소스를 mime형태로 변환합니다.
        var html = DEXT5.getBodyValue('editor1');
        var mimedata = DEXT5.dext5EncodeMime(html, 0);
        if(mimedata == '0'){
            alert('mime 인코딩 오류');
        }
    }

</script>

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">
        new Dext5editor("editor1");
    </script>       
</div>
