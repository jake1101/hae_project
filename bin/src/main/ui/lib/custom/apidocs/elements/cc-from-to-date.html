<cc-from-to-date>

Defined in: html\cc-from-to-date.html:2

from date ~ to date 를 위한 text input 커스텀 태그

<cc-from-to-date from-date="{{from_date}}" to-date={{to_date}}></cc-from-to-date>

Attributes

from-date

시작일자 value binding
[string] [optional]

init-date

fromDate, toDate 기본값 세팅 여부
[boolean] [optional] default=true
- "true" 인 경우, fromDate = "현재날짜 -7", toDate="현재날짜"로 세팅 됨
- "false"인 경우, fromDate, toDate 값 세팅 안함.

init-from-date

fromDate 기본값 설정
(ex) init-from-value="-30" (fromDate로 부터 30일 전으로 fromDate값 세팅)
[number] [optional]

init-to-date

toDate 기본값 설정
[number] [optional]
(ex) init-to-value="30" (toDate로 부터 30일 후로 toDate값 세팅)

time-zone

sc-date-field의 value값 사용을 지정한다
[boolean] [optional] default=false
- "false"인 경우, sc-date-field의 value를 String 값으로 사용 합니다.

to-date

종료일자 value binding
[string] [optional]
